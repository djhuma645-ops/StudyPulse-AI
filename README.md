# 📚 StudyPulse AI

> An all-in-one AI study copilot for students that combines an intent-driven smart fact engine with an automated routine generator to optimize daily learning and exam preparation. 🚀

---

## 🌟 Key Features

### 📅 1. Dynamic Routine Generator
- **Personalized Targets:** Creates custom study plans based on the user's name, class, and chosen subject.
- **Smart Velocity Calculation:** Dynamically computes exactly how many chapters need to be covered per day to complete the syllabus before the exam date.
- **Adaptive Difficulty Analyzer:** Analyzes the user-inputted difficulty level (Easy, Medium, Hard) to generate supportive, contextual action items.
- **Gamified Motivation:** Features an interactive "Topper's Tip" banner that changes to keep students focused and disciplined.

### 🔍 2. Intent-Driven AI (Wiki Search)
- **Direct Fact Extraction:** Extracts precise summaries and featured structural overviews from vast knowledge sources for rapid fact-checking.
- **Semantic Context Analyzer:** Surfaces the underlying contextual history, dates, and related scientific or historical breakthroughs along with the primary answer to give students comprehensive conceptual clarity.
- **Zero Distraction Learning:** Delivers clean information blocks without shifting focus to external, messy web interfaces.

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask Framework) for handling server routing and core logic.
- **Frontend:** Responsive HTML5 with embedded CSS styling (`<style>` tag) for a modern, beautiful pastel UI layout.
